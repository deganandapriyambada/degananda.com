Tools Description
Text Transformator is a tools to transform text with four different transformation options.
  • Capitalize - to modify the captizalization of the text such as UPPERCASE, lowercase, Sentence case and Perfect Case
  • Visualization - add HTML based text visualization modification including bold, italic, underline, strikethrough, superscript, subscript
  • Replacement - Replace character or words with new character/word
  • Enrichment - Add additional character/word before or after specific character/word
