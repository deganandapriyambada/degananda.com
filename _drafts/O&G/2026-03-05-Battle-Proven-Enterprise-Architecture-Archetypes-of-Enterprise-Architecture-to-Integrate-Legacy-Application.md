# The Importance of System Integration on an Organization

Integration is one of the key pre-requisites before undergoing digital transformation project. 

In fact, most of legacy system on any industry that doesnt consider IT (information technology) as their core business process has **same issue/recurring pattern**: decoupled system with non-existent integration where most of the data integration is done manually or the business user will opted to curate / download the data into their local machine and start using microsoft excel perform the computaton.

Such condition will become a great technical debt when digital transformation project is kicked in and usually the first improvement suggested by the appointed consultant/to be system integrator is to integrate the legacy apps.
